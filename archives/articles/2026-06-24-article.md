## 核心内容

### 一、DataFrame：数据操作的核心容器

#### 概念说明
DataFrame是Pandas中的**二维表格数据结构**，类似Excel表格或SQL表。它由行（index）和列（columns）组成，每列可以是不同数据类型（数值、字符串、日期等）。DataFrame是Pandas中最常用的数据结构，几乎所有数据分析操作都围绕它展开。

#### 原理解析
DataFrame底层基于NumPy数组实现，但提供了更高级的标签索引和操作方法。其核心设计思想是**向量化操作**——对整列数据的操作自动循环，无需显式编写for循环。这种设计使代码更简洁、执行速度更快（底层用C语言优化）。

**关键特性**：
- **轴标签**：行索引（index）和列名（columns）提供语义化访问
- **自动对齐**：不同DataFrame运算时会根据行/列标签自动对齐
- **灵活的数据类型**：每列可独立指定dtype（int64, float64, object等）

#### 代码示例

```python
import pandas as pd
import numpy as np

# 创建DataFrame的多种方式
# 方式1：从字典创建（最常用）
data = {
    '姓名': ['张三', '李四', '王五', '赵六'],
    '年龄': [25, 30, 28, 35],
    '薪资': [15000, 20000, 18000, 25000],
    '部门': ['技术', '市场', '技术', '管理']
}
df = pd.DataFrame(data)
print("原始DataFrame:")
print(df)

# 方式2：从CSV文件读取
# df = pd.read_csv('data.csv')

# 基本属性查看
print(f"\n形状: {df.shape}")  # (4, 4)
print(f"列名: {df.columns.tolist()}")
print(f"数据类型:\n{df.dtypes}")
print(f"前2行:\n{df.head(2)}")

# 索引与选择
print("\n选择'薪资'列:")
print(df['薪资'])  # 返回Series

print("\n选择多列:")
print(df[['姓名', '薪资']])

print("\n选择前2行:")
print(df.iloc[:2])  # 位置索引

print("\n选择年龄>28的行:")
print(df[df['年龄'] > 28])  # 布尔索引

# 新增列
df['年薪'] = df['薪资'] * 12
print("\n新增年薪列:")
print(df)

# 删除列
df.drop('年薪', axis=1, inplace=True)
```

### 二、数据清洗：从原始数据到可用数据

#### 概念说明
数据清洗是**识别并修正数据中错误、不完整、不准确或格式不一致部分**的过程。真实世界的数据往往包含缺失值、重复记录、异常值、格式错误等问题，清洗是数据分析前的必要步骤。

#### 原理解析
Pandas将数据质量问题分为三类，并提供对应处理方法：
1. **缺失值**：用NaN表示，处理方法包括删除（dropna）或填充（fillna）
2. **重复值**：完全相同的行，用duplicated检测、drop_duplicates删除
3. **异常值**：统计意义上偏离正常范围的值，通常用IQR或Z-score识别

**核心原则**：清洗操作应**可追溯、可重复**。建议在清洗前备份原始数据，并记录每一步操作。

#### 代码示例

```python
# 创建包含脏数据的示例
dirty_data = {
    '姓名': ['张三', '李四', None, '王五', '张三'],
    '年龄': [25, None, 28, 200, 25],  # 200是异常值
    '薪资': [15000, 20000, None, 18000, 15000],
    '入职日期': ['2023-01-15', '2023/02/20', '2023-03-10', '2023-04-05', '2023-01-15']
}
df_dirty = pd.DataFrame(dirty_data)
print("原始脏数据:")
print(df_dirty)

# 1. 处理缺失值
print("\n缺失值统计:")
print(df_dirty.isnull().sum())

# 方案A：删除包含缺失值的行
df_clean = df_dirty.dropna()
print("\n删除缺失值后:")
print(df_clean)

# 方案B：填充缺失值（更常用）
df_filled = df_dirty.copy()
df_filled['年龄'] = df_filled['年龄'].fillna(df_filled['年龄'].median())  # 用中位数填充
df_filled['薪资'] = df_filled['薪资'].fillna(df_filled['薪资'].mean())    # 用均值填充
df_filled['姓名'] = df_filled['姓名'].fillna('未知')                     # 用固定值填充
print("\n填充缺失值后:")
print(df_filled)

# 2. 处理重复值
print(f"\n重复行数量: {df_filled.duplicated().sum()}")
df_unique = df_filled.drop_duplicates()
print("去重后:")
print(df_unique)

# 3. 处理异常值（使用IQR方法）
Q1 = df_unique['年龄'].quantile(0.25)
Q3 = df_unique['年龄'].quantile(0.75)
IQR = Q3 - Q1
lower_bound = Q1 - 1.5 * IQR
upper_bound = Q3 + 1.5 * IQR

df_no_outlier = df_unique[
    (df_unique['年龄'] >= lower_bound) & 
    (df_unique['年龄'] <= upper_bound)
]
print(f"\n异常值处理（年龄范围: {lower_bound:.0f} ~ {upper_bound:.0f}）:")
print(df_no_outlier)

# 4. 数据类型转换和格式统一
df_no_outlier['入职日期'] = pd.to_datetime(df_no_outlier['入职日期'], errors='coerce')
print("\n日期统一格式后:")
print(df_no_outlier.dtypes)

# 最终清洗结果
print("\n最终清洗完成的数据:")
print(df_no_outlier)
```

### 三、分组聚合：从数据到洞察

#### 概念说明
分组聚合是数据分析的**核心操作模式**，遵循“Split-Apply-Combine”范式：
- **Split**：按某个或多个键将数据分组
- **Apply**：对每组应用函数（聚合、转换、过滤等）
- **Combine**：将结果合并为最终数据结构

#### 原理解析
`groupby`操作创建`GroupBy`对象，该对象**惰性求值**——只有调用聚合方法时才真正计算。理解这一点对性能优化很重要：避免在循环中多次调用groupby，应一次性完成所需操作。

**常用聚合函数**：
- 统计类：`sum`, `mean`, `count`, `std`, `min`, `max`
- 综合类：`agg`（可同时应用多个函数）
- 变换类：`transform`（保持原形状，返回与输入相同长度的结果）

#### 代码示例

```python
# 创建示例数据
sales_data = {
    '城市': ['北京', '上海', '北京', '上海', '广州', '北京', '上海', '广州'],
    '产品': ['A', 'A', 'B', 'B', 'A', 'A', 'B', 'B'],
    '销售额': [100, 150, 80, 120, 90, 110, 130, 70],
    '数量': [10, 15, 8, 12, 9, 11, 13, 7]
}
df_sales = pd.DataFrame(sales_data)
print("销售数据:")
print(df_sales)

# 1. 单列分组聚合
print("\n按城市分组，计算销售额总和:")
city_sum = df_sales.groupby('城市')['销售额'].sum()
print(city_sum)

# 2. 多列分组聚合
print("\n按城市和产品分组，计算销售额和数量的均值:")
grouped = df_sales.groupby(['城市', '产品'])[['销售额', '数量']].mean()
print(grouped)

# 3. 使用agg同时应用多个聚合函数
print("\n使用agg进行多维度统计:")
result = df_sales.groupby('城市').agg({
    '销售额': ['sum', 'mean', 'std'],
    '数量': ['sum', 'count']
})
print(result)

# 4. 使用transform保留原形状
print("\n计算每个城市销售额的百分比贡献:")
df_sales['城市销售占比'] = df_sales.groupby('城市')['销售额'].transform(
    lambda x: x / x.sum() * 100
)
print(df_sales)

# 5. 过滤分组
print("\n筛选总销售额>200的城市:")
filtered = df_sales.groupby('城市').filter(lambda x: x['销售额'].sum() > 200)
print(filtered)

# 6. 实战：销售分析报告
print("\n=== 销售分析报告 ===")
report = df_sales.groupby('城市').agg(
    总销售额=('销售额', 'sum'),
    平均销售额=('销售额', 'mean'),
    总销量=('数量', 'sum'),
    订单数=('销售额', 'count')
).sort_values('总销售额', ascending=False)
print(report)
```

## 实践建议

1. **动手练习：使用真实数据集**  
   立即从Kaggle下载一个CSV数据集（如Titanic或Iris），完整执行“读取→探索→清洗→分组聚合”流程。每天用Pandas处理一个数据集，两周内技能将显著提升。

2. **掌握调试技巧**  
   当代码报错时，使用`df.info()`查看数据类型，`df.describe()`查看统计摘要，`df.isnull().sum()`定位缺失值。养成先探索数据再编写逻辑的习惯。

3. **建立清洗模板**  
   为常见数据问题（缺失值、异常值、重复值）编写可复用的函数库。例如创建一个`clean_data(df)`函数，包含缺失值填充策略、异常值检测逻辑等，提高工作效率。

4. **从“能运行”到“高效运行”**  
   学习使用`pd.read_csv()`的`dtype`参数指定列类型（减少内存），使用`inplace=True`谨慎修改原DataFrame，避免在循环中使用`append`（改用`pd.concat`）。性能优化是高级工程师的必修课。

5. **善用官方文档和社区**  
   Pandas官方文档（pandas.pydata.org）提供详尽的API参考和用户指南。遇到具体操作时，优先搜索“pandas + 你的需求”（如“pandas groupby multiple aggregations”），Stack Overflow有大量高质量解答。
