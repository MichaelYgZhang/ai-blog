# 从零实现一个神经网络：前向传播、反向传播与梯度下降

## 核心结论

**从零实现一个神经网络的核心在于理解并手动实现前向传播（Forward Propagation）、反向传播（Backpropagation）和梯度下降（Gradient Descent）这三个环节，其中反向传播是连接模型预测与参数优化的关键桥梁，而梯度下降则是驱动模型学习的引擎。**

## 为什么重要

在深度学习框架（如PyTorch、TensorFlow）高度自动化的今天，许多初学者容易陷入“调包侠”的困境——能够调用现成API训练模型，却对模型内部的工作原理知之甚少。**从零实现一个神经网络，是打破这种黑箱认知、建立深度学习直觉的最有效途径。** 这个过程迫使你直面每一个神经元的计算细节，理解损失函数如何通过链式法则传递误差，以及梯度如何指导参数更新。

对于一名AI工程师而言，**掌握从零实现的能力意味着你拥有了诊断和优化模型的基础**。当模型不收敛时，你能从梯度的角度分析问题；当需要设计新网络结构时，你能准确预判计算图的可行性。这正是从“使用者”向“创造者”转变的关键一步。本篇文章将带领你手动实现一个仅包含一个隐藏层的二分类神经网络，完整走通从数据输入到参数更新的全过程。

## 核心内容

### 1. 神经网络的计算图与数学基础

在动手编码之前，我们需要建立神经网络的计算模型。一个单隐层神经网络可以看作一个复合函数：**输入层 → 隐藏层（线性变换 + 激活函数）→ 输出层（线性变换 + 激活函数）**。

#### 1.1 数学符号定义

假设我们有一个二分类问题，输入特征维度为 `n_x`，隐藏层神经元数量为 `n_h`，输出层为1个神经元。

- **输入层到隐藏层**：
  - 权重矩阵：`W1`，形状为 `(n_h, n_x)`
  - 偏置向量：`b1`，形状为 `(n_h, 1)`
  - 线性输出：`Z1 = W1 @ X + b1`
  - 激活后输出：`A1 = ReLU(Z1)` 或 `tanh(Z1)`

- **隐藏层到输出层**：
  - 权重矩阵：`W2`，形状为 `(1, n_h)`
  - 偏置向量：`b2`，形状为 `(1, 1)`
  - 线性输出：`Z2 = W2 @ A1 + b2`
  - 最终输出：`A2 = sigmoid(Z2)`

其中 `X` 是输入数据矩阵，形状为 `(n_x, m)`，`m` 是样本数量。**这里的核心思想是：每一层都在做“线性变换 + 非线性映射”，通过堆叠这样的层，网络可以学习到极其复杂的函数。**

#### 1.2 损失函数

对于二分类问题，我们使用交叉熵损失函数（Binary Cross-Entropy）：

`L = - (1/m) * Σ [ y * log(A2) + (1 - y) * log(1 - A2) ]`

这个损失函数衡量了预测分布 `A2` 与真实标签 `y` 之间的差异，**值越小表示预测越准确**。

### 2. 前向传播：计算预测值

前向传播就是按照计算图的顺序，从输入 `X` 开始，逐层计算并存储中间结果，最终得到预测值 `A2`。**这个过程本质上是函数求值。**

```python
import numpy as np

def initialize_parameters(n_x, n_h, n_y):
    """
    初始化网络参数
    """
    np.random.seed(42)
    W1 = np.random.randn(n_h, n_x) * 0.01
    b1 = np.zeros((n_h, 1))
    W2 = np.random.randn(n_y, n_h) * 0.01
    b2 = np.zeros((n_y, 1))
    
    parameters = {
        "W1": W1,
        "b1": b1,
        "W2": W2,
        "b2": b2
    }
    return parameters

def forward_propagation(X, parameters):
    """
    前向传播
    X -- 输入数据，形状 (n_x, m)
    返回：
    A2 -- 最终输出（预测概率）
    cache -- 存储中间变量，用于反向传播
    """
    W1 = parameters["W1"]
    b1 = parameters["b1"]
    W2 = parameters["W2"]
    b2 = parameters["b2"]
    
    # 隐藏层
    Z1 = np.dot(W1, X) + b1
    A1 = np.tanh(Z1)  # 使用tanh激活函数
    
    # 输出层
    Z2 = np.dot(W2, A1) + b2
    A2 = 1 / (1 + np.exp(-Z2))  # sigmoid激活函数
    
    cache = {
        "Z1": Z1,
        "A1": A1,
        "Z2": Z2,
        "A2": A2
    }
    return A2, cache
```

**关键要点**：
- 权重用小的随机数初始化（0.01倍），打破对称性
- 偏置初始化为0是可行的
- **务必缓存中间变量 `Z1, A1, Z2`，它们是反向传播的输入**

### 3. 反向传播：计算梯度

反向传播是神经网络训练中最精妙的部分。**它的核心是链式法则（Chain Rule）**：从输出层的损失开始，逐层向前计算每个参数对损失的偏导数（梯度）。

#### 3.1 输出层的梯度计算

对于输出层，我们计算 `dZ2`，即损失 `L` 对 `Z2` 的导数。对于sigmoid + 交叉熵的组合，有一个简洁的表达式：

`dZ2 = A2 - Y`

其中 `Y` 是真实标签矩阵。**这个简洁的结果是数学推导的优美体现**，它极大地简化了实现。

#### 3.2 隐藏层的梯度计算

从 `dZ2` 出发，利用链式法则逐步计算：

1. `dW2 = (1/m) * dZ2 @ A1.T`
2. `db2 = (1/m) * np.sum(dZ2, axis=1, keepdims=True)`
3. `dA1 = W2.T @ dZ2`
4. `dZ1 = dA1 * (1 - np.power(A1, 2))`  # tanh的导数
5. `dW1 = (1/m) * dZ1 @ X.T`
6. `db1 = (1/m) * np.sum(dZ1, axis=1, keepdims=True)`

**理解梯度流动的方向至关重要**：`dZ2` 的梯度通过 `W2` 流向 `A1`，然后通过激活函数流向 `Z1`，最终流向 `W1` 和 `b1`。

```python
def backward_propagation(parameters, cache, X, Y):
    """
    反向传播
    """
    m = X.shape[1]
    W1 = parameters["W1"]
    W2 = parameters["W2"]
    
    A1 = cache["A1"]
    A2 = cache["A2"]
    
    # 输出层梯度
    dZ2 = A2 - Y
    dW2 = (1/m) * np.dot(dZ2, A1.T)
    db2 = (1/m) * np.sum(dZ2, axis=1, keepdims=True)
    
    # 隐藏层梯度
    dA1 = np.dot(W2.T, dZ2)
    dZ1 = dA1 * (1 - np.power(A1, 2))  # tanh导数
    dW1 = (1/m) * np.dot(dZ1, X.T)
    db1 = (1/m) * np.sum(dZ1, axis=1, keepdims=True)
    
    grads = {
        "dW1": dW1,
        "db1": db1,
        "dW2": dW2,
        "db2": db2
    }
    return grads
```

### 4. 梯度下降：更新参数

有了梯度之后，参数更新就变得直接了。**梯度下降的基本思想是：沿着梯度的反方向更新参数，使损失函数值减小。**

`W = W - learning_rate * dW`
`b = b - learning_rate * db`

学习率 `learning_rate` 是一个超参数，控制每一步更新的步长。**步长太小收敛慢，步太大会导致震荡甚至发散。**

```python
def update_parameters(parameters, grads, learning_rate=1.2):
    """
    使用梯度下降更新参数
    """
    parameters["W1"] -= learning_rate * grads["dW1"]
    parameters["b1"] -= learning_rate * grads["db1"]
    parameters["W2"] -= learning_rate * grads["dW2"]
    parameters["b2"] -= learning_rate * grads["db2"]
    
    return parameters
```

### 5. 整合训练循环

将上述所有组件整合到一个训练函数中，形成完整的训练流程：

```python
def train(X, Y, n_h, num_iterations=10000, learning_rate=1.2):
    """
    完整训练流程
    """
    n_x = X.shape[0]
    n_y = Y.shape[0]
    
    # 初始化参数
    parameters = initialize_parameters(n_x, n_h, n_y)
    
    for i in range(num_iterations):
        # 前向传播
        A2, cache = forward_propagation(X, parameters)
        
        # 计算损失
        cost = - (1/m) * np.sum(Y * np.log(A2) + (1 - Y) * np.log(1 - A2))
        
        # 反向传播
        grads = backward_propagation(parameters, cache, X, Y)
        
        # 更新参数
        parameters = update_parameters(parameters, grads, learning_rate)
        
        # 打印损失
        if i % 1000 == 0:
            print(f"迭代 {i}: 损失 = {cost}")
    
    return parameters
```

**训练过程的本质就是反复执行：前向传播 → 计算损失 → 反向传播 → 参数更新**。这个循环每执行一次，模型就向最优参数迈进一小步。

## 知识依赖说明

理解本文需要以下前置知识：

1. **微积分基础**：特别是**链式法则**。反向传播的本质就是链式法则的反复应用。如果你对复合函数求导感到生疏，建议先复习一元函数和多元函数的链式法则。
2. **线性代数**：矩阵乘法、矩阵转置、矩阵求导。本文中的 `np.dot` 操作和梯度推导都依赖线性代数知识。
3. **Python编程**：NumPy库的基本操作，包括数组创建、广播机制、矩阵运算等。
4. **概率论基础**：交叉熵损失函数来源于信息论和概率统计，理解它的含义有助于把握模型优化的目标。

建议在阅读本文前，先回顾阶段1中的“微积分”和“线性代数”模块，特别是矩阵求导的相关内容。

## 实践建议

1. **动手逐行实现**：不要复制粘贴代码。手动敲一遍每一行代码，在关键位置添加 `print` 语句查看张量形状和数值变化，这会极大加深你的理解。

2. **用玩具数据集验证**：使用 sklearn 的 `make_moons` 或 `make_circles` 生成简单的二分类数据集，训练你的网络并可视化决策边界。**观察不同隐藏层神经元数量对决策边界形状的影响。**

3. **调试反向传播**：实现梯度检查（Gradient Checking）——用数值方法近似计算梯度，与你的反向传播结果对比。**这是发现反向传播代码bug的最可靠方法。**

4. **尝试不同激活函数**：将隐藏层的 `tanh` 替换为 `ReLU`（`np.maximum(0, Z1)`），观察训练速度和最终准确率的变化。**注意ReLU的导数计算方式不同。**

5. **调整超参数**：固定其他条件，分别改变学习率和隐藏层大小，观察损失下降曲线的变化。**记录不同超参数组合下的训练结果，建立调参的直觉。**
