# 线性回归与逻辑回归从零实现

**核心结论：线性回归与逻辑回归是监督学习的两个基石模型，前者通过最小二乘和梯度下降解决连续值预测，后者通过引入Sigmoid函数将线性输出映射为概率，从而解决分类问题；从零实现这两个模型，是理解深度学习从“线性变换”到“非线性激活”的关键一步。**

## 为什么重要

在AI工程师的成长路径中，线性回归和逻辑回归是连接“数学原理”与“工程实践”的第一座桥梁。许多初学者直接跳入复杂的神经网络，却常常在反向传播、损失函数设计等基础概念上栽跟头。回归和逻辑回归的价值在于：**它们用最简洁的数学形式，展示了机器学习模型的核心范式——定义假设函数、设计损失函数、通过优化算法求解参数。**

从工程角度看，这两个模型是工业界最常用的基线模型。无论是金融领域的信用评分（逻辑回归），还是推荐系统中的价格预测（线性回归），它们都以计算效率高、可解释性强著称。更重要的是，**理解线性回归的梯度下降过程，就理解了神经网络反向传播的本质；理解逻辑回归的Sigmoid函数，就理解了为什么深度学习需要激活函数来引入非线性。**

作为AI工程师，如果你能徒手写出这两个模型的训练循环，你便掌握了机器学习最底层的“肌肉记忆”——这比任何高级框架的API调用都更有价值。

## 核心内容

### 1. 线性回归：从最小二乘到梯度下降

#### 概念说明

**线性回归**假设目标值 \( y \) 与特征 \( x_1, x_2, \ldots, x_n \) 之间存在线性关系：  
\[
\hat{y} = w_1 x_1 + w_2 x_2 + \cdots + w_n x_n + b
\]

其中 \( w \) 是权重，\( b \) 是偏置。我们的目标是找到一组参数 \( (w, b) \)，使得预测值 \( \hat{y} \) 与真实值 \( y \) 之间的误差最小。

#### 原理解析

**最小二乘法**是最直观的优化目标：**最小化所有样本预测误差的平方和**。其损失函数（均方误差，MSE）定义为：
\[
L(w, b) = \frac{1}{2m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)})^2
\]

为什么要加 \( \frac{1}{2} \)？这是为了在求导时消去平方项带来的系数2，简化计算。

**梯度下降**是求解该优化问题的核心算法。其思想是：沿着损失函数下降最快的方向（即负梯度方向）更新参数：
\[
w_j := w_j - \alpha \frac{\partial L}{\partial w_j}
\]

对MSE求偏导可得：
\[
\frac{\partial L}{\partial w_j} = \frac{1}{m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)}) x_j^{(i)}
\]

这就是**参数更新的核心公式**：误差乘以特征值，再累加平均。

#### 代码实现

```python
import numpy as np

class LinearRegression:
    def __init__(self, lr=0.01, epochs=1000):
        self.lr = lr
        self.epochs = epochs
        self.w = None
        self.b = None
    
    def fit(self, X, y):
        m, n = X.shape
        self.w = np.zeros(n)
        self.b = 0
        
        for epoch in range(self.epochs):
            # 前向传播：计算预测值
            y_pred = np.dot(X, self.w) + self.b
            
            # 计算梯度
            dw = (1/m) * np.dot(X.T, (y_pred - y))
            db = (1/m) * np.sum(y_pred - y)
            
            # 参数更新
            self.w -= self.lr * dw
            self.b -= self.lr * db
            
            # 每隔100轮打印损失
            if epoch % 100 == 0:
                loss = np.mean((y_pred - y)**2) / 2
                print(f"Epoch {epoch}, Loss: {loss:.4f}")
    
    def predict(self, X):
        return np.dot(X, self.w) + self.b

# 使用示例
if __name__ == "__main__":
    # 生成模拟数据：y = 2*x1 + 3*x2 + 1 + 噪声
    np.random.seed(42)
    X = np.random.randn(100, 2)
    true_w = np.array([2.0, 3.0])
    true_b = 1.0
    y = np.dot(X, true_w) + true_b + 0.1 * np.random.randn(100)
    
    model = LinearRegression(lr=0.1, epochs=1000)
    model.fit(X, y)
    print(f"训练得到的权重: {model.w}")
    print(f"训练得到的偏置: {model.b:.4f}")
```

**关键洞察**：上述代码中，`dw`的计算等价于`(1/m) * X.T @ (y_pred - y)`。这正是线性回归梯度下降的向量化实现，也是深度学习反向传播中`dW = X.T @ dZ`的原型。

### 2. 逻辑回归：从线性到非线性的跃迁

#### 概念说明

**逻辑回归**虽然名字带“回归”，实际解决的是**二分类问题**。其核心思想是：**先用线性模型计算一个分数，再通过Sigmoid函数将其压缩到[0,1]区间，作为属于正类的概率**。

数学表达为：
\[
z = w^T x + b
\]
\[
\hat{y} = \sigma(z) = \frac{1}{1 + e^{-z}}
\]

#### 原理解析

**Sigmoid函数** \( \sigma(z) \) 是逻辑回归的灵魂。它有三个关键性质：
- **输出范围(0,1)**：天然适合表示概率
- **单调递增**：z越大，概率越高
- **可导且导数优美**：\( \sigma'(z) = \sigma(z)(1 - \sigma(z)) \)

逻辑回归不适用MSE作为损失函数，因为MSE在分类任务中会导致非凸优化问题。取而代之的是**交叉熵损失（对数损失）**：
\[
L = -\frac{1}{m} \sum_{i=1}^{m} [y^{(i)} \log(\hat{y}^{(i)}) + (1 - y^{(i)}) \log(1 - \hat{y}^{(i)})]
\]

这个损失函数的直观理解：当真实标签 \( y=1 \) 时，我们希望 \( \hat{y} \) 接近1（即 \( \log(\hat{y}) \) 接近0）；当 \( y=0 \) 时，我们希望 \( \hat{y} \) 接近0（即 \( \log(1-\hat{y}) \) 接近0）。

对交叉熵损失求导，可以得到与线性回归**形式完全一致**的梯度：
\[
\frac{\partial L}{\partial w_j} = \frac{1}{m} \sum_{i=1}^{m} (\hat{y}^{(i)} - y^{(i)}) x_j^{(i)}
\]

唯一的区别是：这里的 \( \hat{y} \) 经过了Sigmoid变换。**这就是为什么说逻辑回归只是在线性回归的输出层加了一个Sigmoid激活函数**。

#### 代码实现

```python
import numpy as np

class LogisticRegression:
    def __init__(self, lr=0.01, epochs=1000):
        self.lr = lr
        self.epochs = epochs
        self.w = None
        self.b = None
    
    def sigmoid(self, z):
        # 防止数值溢出：当z很大时，e^{-z}可能下溢
        # 使用np.clip确保数值稳定性
        z = np.clip(z, -500, 500)
        return 1 / (1 + np.exp(-z))
    
    def fit(self, X, y):
        m, n = X.shape
        self.w = np.zeros(n)
        self.b = 0
        
        for epoch in range(self.epochs):
            # 前向传播
            z = np.dot(X, self.w) + self.b
            y_pred = self.sigmoid(z)
            
            # 计算梯度（与线性回归完全一致！）
            dw = (1/m) * np.dot(X.T, (y_pred - y))
            db = (1/m) * np.sum(y_pred - y)
            
            # 参数更新
            self.w -= self.lr * dw
            self.b -= self.lr * db
            
            # 打印交叉熵损失
            if epoch % 100 == 0:
                # 防止log(0)
                y_pred_clipped = np.clip(y_pred, 1e-12, 1 - 1e-12)
                loss = -np.mean(y * np.log(y_pred_clipped) + 
                               (1 - y) * np.log(1 - y_pred_clipped))
                print(f"Epoch {epoch}, Loss: {loss:.4f}")
    
    def predict_proba(self, X):
        z = np.dot(X, self.w) + self.b
        return self.sigmoid(z)
    
    def predict(self, X, threshold=0.5):
        proba = self.predict_proba(X)
        return (proba >= threshold).astype(int)

# 使用示例
if __name__ == "__main__":
    # 生成二分类数据
    np.random.seed(42)
    from sklearn.datasets import make_classification
    X, y = make_classification(n_samples=200, n_features=2, 
                               n_redundant=0, random_state=42)
    
    model = LogisticRegression(lr=0.1, epochs=1000)
    model.fit(X, y)
    
    # 预测并评估
    y_pred = model.predict(X)
    accuracy = np.mean(y_pred == y)
    print(f"训练准确率: {accuracy:.4f}")
```

**关键洞察**：比较线性回归和逻辑回归的代码，你会发现**梯度更新公式完全相同**。区别仅在于：
1. 逻辑回归在前向传播中增加了Sigmoid激活
2. 逻辑回归使用交叉熵损失而非MSE
3. 逻辑回归的输出需要阈值化才能得到类别标签

这正是深度学习“层”的概念雏形：**线性层 + 激活函数 = 一个基本网络块**。

### 3. 从零到框架：理解深度学习的基础抽象

通过上述实现，我们可以提炼出深度学习训练的**通用范式**：

```
1. 前向传播：计算预测值 (y_pred = activation(X @ w + b))
2. 计算损失：比较预测值与真实值 (loss = loss_function(y_pred, y))
3. 反向传播：计算梯度 (dw, db)
4. 参数更新：沿梯度下降 (w -= lr * dw, b -= lr * db)
```

PyTorch等框架所做的，就是**自动化这个过程**。当你调用`loss.backward()`时，框架自动计算所有参数的梯度；当你调用`optimizer.step()`时，框架自动更新参数。理解上述从零实现，你就理解了框架背后的一切。

## 知识依赖说明

理解本文需要以下前置知识：

1. **线性代数基础**：特别是矩阵乘法（`np.dot`的数学含义）、向量化运算。这是理解梯度公式中 `X.T @ (y_pred - y)` 的关键。
2. **微积分基础**：偏导数、链式法则。梯度下降的核心就是求偏导，而深度学习反向传播的本质就是链式法则的递归应用。
3. **Python基础**：NumPy数组操作、面向对象编程。上述代码使用了类封装和向量化计算，这是高效实现机器学习模型的基础。

如果你对这些知识感到陌生，建议先回顾阶段1「编程与数学基础」中的线性代数部分，特别是矩阵运算章节。

## 实践建议

1. **动手复现并调试**：不要仅仅阅读代码，打开Jupyter Notebook逐行运行。故意修改学习率（如设为0.5或0.0001），观察损失曲线如何变化。**理解“学习率过大导致震荡、过小导致收敛慢”的直观现象**。

2. **添加正则化**：在损失函数中加入L2正则项（权重衰减），实现岭回归和L2正则化的逻辑回归。观察正则化系数对模型复杂度的影响。

3. **扩展到多分类**：将逻辑回归的Sigmoid替换为Softmax函数，实现多分类逻辑回归（Softmax回归）。你会发现，**Softmax回归与神经网络的输出层完全等价**。

4. **对比Mini-batch梯度下降**：将上述代码改为每次随机采样32个样本更新参数。对比全批量梯度下降（Batch GD）与随机梯度下降（SGD）的收敛曲线差异。

5. **可视化决策边界**：对二维数据训练模型后，绘制模型的决策边界。观察线性回归的决策边界是一条直线，逻辑回归的决策边界是一条直线（因为决策边界由 \( w^T x + b = 0 \) 定义）。**这解释了为什么逻辑回归只能解决线性可分问题**。
