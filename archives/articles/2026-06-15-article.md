# 降维技术：PCA与t-SNE

## 核心结论

**PCA和t-SNE是两种核心的降维技术，PCA通过线性变换保留数据的全局方差结构，适用于特征压缩和去噪；t-SNE通过非线性映射保留数据的局部邻域关系，专注于高维数据的可视化，两者在AI工程中分别服务于“特征工程”和“探索性分析”两大核心场景。**

---

## 为什么重要

在AI工程师的成长路径中，降维技术是连接“数据处理”与“模型优化”的关键桥梁。当我们面对高维数据时，**维度灾难**会导致计算成本激增、模型过拟合、可视化困难等一系列问题。例如，一张256×256的灰度图像有65536个特征，直接输入模型不仅训练缓慢，还会因为特征稀疏导致梯度不稳定。降维技术能够从高维空间中提取出最具有信息量的低维表示，让模型专注于“真正重要”的模式，同时显著降低计算复杂度。

从能力进阶的角度看，掌握PCA和t-SNE意味着你具备了两种不同维度的思考能力：PCA教会你**用线性代数视角理解数据方差**，这是特征工程和模型解释性的基础；t-SNE则训练你**用概率和流形学习视角理解数据局部结构**，这是深度学习中表征学习和可视化分析的核心。两者结合，你就能在特征压缩、噪声过滤、数据探索、结果可视化等场景中游刃有余。

此外，**PCA和t-SNE代表了降维技术的两大类别：线性与非线性**。理解它们的原理和差异，为你后续学习自编码器、UMAP、LEM等更高级的降维方法打下坚实基础。这不仅是工具的使用，更是对数据内在结构的认知升级。

## 核心内容

### PCA：主成分分析

#### 概念说明

PCA（Principal Component Analysis）是一种**线性降维**技术，它通过寻找一组正交的“主成分”方向，将原始高维数据投影到这些方向上，使得投影后的数据方差最大化。简单来说，PCA找到数据“最分散”的方向，认为这些方向包含了最重要的信息。

#### 原理解析

PCA的核心是**特征值分解**。假设我们有n个样本、每个样本有d个特征，构成数据矩阵X（n×d）。PCA的步骤如下：

1. **数据中心化**：对每个特征减去均值，得到零均值矩阵X_centered。这一步确保第一主成分通过原点。
2. **计算协方差矩阵**：C = (1/(n-1)) * X_centered^T * X_centered，C是一个d×d的对称矩阵，对角线元素是各特征的方差，非对角线元素是特征间的协方差。
3. **特征值分解**：对C进行特征值分解，得到特征值λ₁≥λ₂≥...≥λ_d和对应的特征向量v₁, v₂, ..., v_d。**特征值越大，对应方向上的方差越大**，该方向携带的信息越多。
4. **选择主成分**：根据累计方差贡献率（前k个特征值之和占总特征值之和的比例）选择前k个特征向量作为新的基。通常保留90%~95%的方差。
5. **数据投影**：将原始数据投影到选定的k个特征向量上，得到降维后的数据Y = X_centered * V_k，其中V_k是前k个特征向量组成的矩阵。

**数学本质**：PCA其实是在寻找一个正交变换矩阵W，使得变换后的数据Y = XW的协方差矩阵是对角矩阵，且对角线元素（方差）从大到小排列。这等价于在约束W^T W = I下最大化Tr(W^T X^T X W)。

#### 代码示例

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import load_iris
from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler

# 加载数据
iris = load_iris()
X = iris.data  # 150个样本，4个特征
y = iris.target

# 标准化（PCA对尺度敏感）
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 手动实现PCA
def pca_manual(X, n_components):
    # 中心化
    X_centered = X - np.mean(X, axis=0)
    # 协方差矩阵
    cov_matrix = np.cov(X_centered, rowvar=False)
    # 特征值分解
    eigenvalues, eigenvectors = np.linalg.eigh(cov_matrix)
    # 按特征值降序排列
    idx = np.argsort(eigenvalues)[::-1]
    eigenvalues = eigenvalues[idx]
    eigenvectors = eigenvectors[:, idx]
    # 选择前n_components个特征向量
    V = eigenvectors[:, :n_components]
    # 投影
    X_reduced = X_centered @ V
    return X_reduced, eigenvalues, V

X_pca_manual, eigenvals, V = pca_manual(X_scaled, 2)

# 使用sklearn验证
pca_sklearn = PCA(n_components=2)
X_pca_sklearn = pca_sklearn.fit_transform(X_scaled)

# 可视化
plt.figure(figsize=(12, 5))
plt.subplot(1, 2, 1)
plt.scatter(X_pca_manual[:, 0], X_pca_manual[:, 1], c=y, cmap='viridis')
plt.title('Manual PCA')
plt.xlabel('PC1')
plt.ylabel('PC2')

plt.subplot(1, 2, 2)
plt.scatter(X_pca_sklearn[:, 0], X_pca_sklearn[:, 1], c=y, cmap='viridis')
plt.title('Sklearn PCA')
plt.xlabel('PC1')
plt.ylabel('PC2')

# 打印方差解释率
print(f"手动PCA方差解释率: {eigenvals[:2].sum() / eigenvals.sum():.3f}")
print(f"Sklearn方差解释率: {pca_sklearn.explained_variance_ratio_.sum():.3f}")
plt.show()
```

**实践指导**：在实际项目中，PCA主要用于：
- **特征降维**：将高维特征压缩到较低维度，减少过拟合风险
- **数据可视化**：将数据降到2D或3D进行可视化
- **噪声过滤**：只保留前k个主成分，丢弃方差小的成分（通常是噪声）

### t-SNE：t分布随机邻域嵌入

#### 概念说明

t-SNE（t-distributed Stochastic Neighbor Embedding）是一种**非线性降维**技术，专注于将高维数据映射到低维空间（通常是2D或3D），同时**保持数据点之间的局部相似性**。它特别擅长揭示高维数据中的聚类结构和流形结构，是深度学习可视化分析的标准工具。

#### 原理解析

t-SNE的核心思想是：**在高维空间中相似的样本，在低维空间中应该仍然相似**。它通过概率分布来建模这种相似性：

1. **高维空间的相似性**：对于每个数据点xᵢ，计算它与其他所有点xⱼ的条件概率p_{j|i}，表示xᵢ选择xⱼ作为邻居的概率。这个概率基于高斯分布，带宽σᵢ由**困惑度（perplexity）**参数控制。困惑度可以理解为“有效邻居数”，通常取值5-50。

2. **低维空间的相似性**：对于降维后的对应点yᵢ和yⱼ，用**t分布**（自由度1）计算相似性q_{ij}。t分布有更重的尾部，这意味着在低维空间中，**不相似的点会被推得更远**，从而缓解“拥挤问题”。

3. **优化目标**：最小化两个概率分布之间的**KL散度**。直观上，如果两个点在低维空间的距离与高维空间不匹配，KL散度就会惩罚这种不匹配。优化通过梯度下降完成，每次迭代调整低维嵌入点的位置。

**关键参数**：
- **perplexity**：控制邻居数量的平衡，影响局部与全局结构的权衡
- **learning_rate**：梯度下降的学习率，通常500-1000
- **n_iter**：迭代次数，通常1000-5000

#### 代码示例

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.datasets import load_digits
from sklearn.manifold import TSNE
from sklearn.decomposition import PCA

# 加载手写数字数据集（8x8像素，64维）
digits = load_digits()
X = digits.data  # 1797个样本，64个特征
y = digits.target

# 先用PCA降维到30维，减少t-SNE的计算量
pca = PCA(n_components=30)
X_pca = pca.fit_transform(X)

# t-SNE降维到2D
tsne = TSNE(n_components=2, perplexity=30, learning_rate=200, random_state=42)
X_tsne = tsne.fit_transform(X_pca)

# 可视化
plt.figure(figsize=(10, 8))
scatter = plt.scatter(X_tsne[:, 0], X_tsne[:, 1], c=y, cmap='tab10', alpha=0.7)
plt.colorbar(scatter)
plt.title('t-SNE Visualization of Digits Dataset (perplexity=30)')
plt.xlabel('t-SNE Component 1')
plt.ylabel('t-SNE Component 2')

# 不同perplexity的对比
fig, axes = plt.subplots(2, 2, figsize=(12, 12))
perplexities = [5, 15, 30, 50]
for ax, perp in zip(axes.ravel(), perplexities):
    tsne_temp = TSNE(n_components=2, perplexity=perp, random_state=42)
    X_temp = tsne_temp.fit_transform(X_pca)
    ax.scatter(X_temp[:, 0], X_temp[:, 1], c=y, cmap='tab10', alpha=0.7)
    ax.set_title(f'perplexity={perp}')
plt.tight_layout()
plt.show()
```

**实践指导**：使用t-SNE时需注意：
- **结果随机性**：每次运行结果不同，建议固定random_state
- **计算复杂度**：O(n²)，大数据集建议先用PCA降维
- **不能用于新样本**：t-SNE没有显式的映射函数，无法处理新数据
- **距离解释性差**：簇之间的距离没有实际意义，关注的是相对位置

### PCA vs t-SNE：关键对比

| 维度 | PCA | t-SNE |
|------|-----|-------|
| **类型** | 线性 | 非线性 |
| **目标** | 保留全局方差 | 保留局部邻域 |
| **输出** | 可解释的主成分 | 不可解释的嵌入坐标 |
| **计算复杂度** | O(d²n + d³) | O(n²) |
| **可逆性** | 可逆（可重建） | 不可逆 |
| **适用场景** | 特征压缩、去噪 | 可视化、聚类发现 |
| **超参数** | 主成分数k | perplexity, learning_rate |
| **随机性** | 确定性 | 随机性 |

## 实践建议

1. **先PCA后t-SNE**：在处理高维数据（>50维）时，先用PCA降到30-50维，再使用t-SNE。这能显著减少t-SNE的计算时间，同时去除噪声，让t-SNE的聚类效果更好。

2. **标准化是PCA的前提**：PCA对特征的尺度敏感。如果特征的量纲不同（如年龄0-100，收入0-100000），必须进行标准化（Z-score或Min-Max），否则方差大的特征会主导主成分。

3. **t-SNE参数调优从perplexity=30开始**：这是最常用的默认值。如果数据点较少（<100），降低perplexity；如果数据点较多（>10000），增加perplexity。同时观察不同perplexity下的结果稳定性。

4. **用方差解释率指导PCA维度选择**：绘制“累计方差解释率 vs 主成分数”的曲线，选择拐点处或达到90%方差时的维度。不要盲目选择2D或3D，除非是为了可视化。

5. **t-SNE结果不要过度解读**：t-SNE的簇大小、簇间距离没有绝对意义。多次运行观察稳定模式，结合其他方法（如聚类算法）验证发现的模式。
