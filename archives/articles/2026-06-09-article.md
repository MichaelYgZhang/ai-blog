# SVM支持向量机原理与实战

## 核心结论

**支持向量机（SVM）通过寻找最大间隔超平面来实现最优分类，其核心思想是将分类问题转化为一个凸二次规划问题，并借助核函数将低维线性不可分数据映射到高维空间，从而在理论上保证了全局最优解和强大的泛化能力。**

## 为什么重要

在机器学习领域，SVM被誉为“最优雅的分类器之一”。它不仅在理论层面完美融合了**线性代数**、**凸优化**和**统计学习理论**，更在实践中展现出惊人的鲁棒性。对于AI工程师而言，掌握SVM意味着：

**第一，理解分类问题的本质。** 与神经网络“黑箱”式的学习不同，SVM通过明确的几何意义（最大间隔）和数学推导（对偶问题）揭示了分类器如何做出决策。这种透明性对于构建可解释的AI系统至关重要。

**第二，为深度学习打下基础。** SVM中的核技巧（Kernel Trick）思想——通过非线性变换将低维问题映射到高维解决——与深度学习中特征表示学习有异曲同工之妙。理解SVM的对偶问题，也为理解支持向量回归（SVR）和结构化SVM等高级模型铺平了道路。

**第三，解决小样本高维问题。** 在文本分类、生物信息学等特征维度远大于样本数量的场景中，SVM凭借其最大间隔原则和结构风险最小化理论，往往比深度神经网络表现更优。

## 核心内容

### 1. 最大间隔：分类器的几何直觉

#### 概念说明

**最大间隔**是SVM最直观的核心思想。对于二分类问题，我们需要找到一个超平面将两类样本分开。但存在无数个这样的超平面——SVM选择的是那个**离最近样本点（支持向量）距离最大**的超平面。

这个间隔（Margin）定义为：超平面到两侧最近样本点的距离之和。**间隔越大，分类器的鲁棒性越强，泛化误差的上界越低。**

#### 原理解析

从线性代数角度看，超平面由法向量 **w** 和偏置 **b** 定义：**w·x + b = 0**。样本点 **x** 到超平面的几何距离为：

\[
\text{distance} = \frac{|w·x + b|}{\|w\|}
\]

对于线性可分数据，我们希望所有正类样本满足 **w·x + b ≥ 1**，负类样本满足 **w·x + b ≤ -1**。此时，间隔大小为 **2 / ‖w‖**。最大化间隔等价于最小化 **‖w‖² / 2**。

**关键洞察**：最终决策只由少数“支持向量”（即恰好落在间隔边界上的样本点）决定，其他样本点对模型没有影响。这体现了SVM的稀疏性。

#### 代码示例

```python
import numpy as np
from sklearn.svm import SVC
import matplotlib.pyplot as plt

# 生成线性可分数据
np.random.seed(42)
X = np.random.randn(100, 2)
y = np.where(X[:, 0] + X[:, 1] > 0, 1, -1)

# 训练线性SVM
model = SVC(kernel='linear', C=1e5)  # 大C表示硬间隔
model.fit(X, y)

# 可视化决策边界和间隔
def plot_svm_decision_boundary(model, X, y):
    # 获取支持向量
    support_vectors = model.support_vectors_
    
    # 创建网格
    x_min, x_max = X[:, 0].min() - 1, X[:, 0].max() + 1
    y_min, y_max = X[:, 1].min() - 1, X[:, 1].max() + 1
    xx, yy = np.meshgrid(np.linspace(x_min, x_max, 100),
                         np.linspace(y_min, y_max, 100))
    
    # 预测网格点
    Z = model.decision_function(np.c_[xx.ravel(), yy.ravel()])
    Z = Z.reshape(xx.shape)
    
    # 绘制
    plt.contourf(xx, yy, Z, levels=[-1, 0, 1], alpha=0.3, colors=['blue', 'gray', 'red'])
    plt.scatter(X[:, 0], X[:, 1], c=y, cmap='bwr', edgecolors='k')
    plt.scatter(support_vectors[:, 0], support_vectors[:, 1], 
                s=100, facecolors='none', edgecolors='k', linewidths=2, label='Support Vectors')
    plt.legend()
    plt.title('SVM with Maximum Margin')
    plt.show()

plot_svm_decision_boundary(model, X, y)
print(f"支持向量数量: {len(model.support_vectors_)}")
```

### 2. 对偶问题：从几何到优化的转换

#### 概念说明

直接求解 **min ‖w‖²/2** 受约束于 **y_i(w·x_i + b) ≥ 1** 是一个**凸二次规划**问题，但引入拉格朗日乘子法后，我们可以将其转化为**对偶问题**。**对偶问题的核心优势**在于：它将优化变量从 **w**（维度等于特征数）转换为 **α**（维度等于样本数），从而让核技巧成为可能。

#### 原理解析

构建拉格朗日函数：

\[
L(w, b, α) = \frac{1}{2}‖w‖² - \sum_{i=1}^n α_i [y_i(w·x_i + b) - 1]
\]

其中 **α_i ≥ 0** 是拉格朗日乘子。通过对 **w** 和 **b** 求偏导并令其为零，得到：

- **w** = Σ α_i y_i x_i
- Σ α_i y_i = 0

代入原函数得到对偶问题：

\[
\max_{α} \sum_{i=1}^n α_i - \frac{1}{2} \sum_{i=1}^n \sum_{j=1}^n α_i α_j y_i y_j (x_i · x_j)
\]

约束条件：α_i ≥ 0，且 Σ α_i y_i = 0。

**关键洞察**：对偶问题中，数据以 **内积形式 (x_i · x_j)** 出现。这正是核技巧的入口——我们可以用核函数 **K(x_i, x_j)** 替换内积，实现隐式的高维映射。

#### 代码示例：手动实现对偶求解（简化版）

```python
import numpy as np
from scipy.optimize import minimize

def dual_objective(alpha, X, y):
    """对偶目标函数（负号用于最小化）"""
    n = len(y)
    # 计算 Σα_i
    sum_alpha = np.sum(alpha)
    # 计算 ΣΣ α_i α_j y_i y_j (x_i·x_j)
    kernel_matrix = np.dot(X, X.T)
    quad_term = np.sum(np.outer(alpha * y, alpha * y) * kernel_matrix)
    return -(sum_alpha - 0.5 * quad_term)

def train_svm_dual(X, y, C=1.0):
    """使用对偶方法训练SVM"""
    n = len(y)
    # 初始猜测
    alpha0 = np.zeros(n)
    # 约束：0 ≤ α_i ≤ C
    bounds = [(0, C) for _ in range(n)]
    # 约束：Σ α_i y_i = 0
    constraints = [{'type': 'eq', 'fun': lambda a: np.dot(a, y)}]
    
    result = minimize(dual_objective, alpha0, args=(X, y), 
                      bounds=bounds, constraints=constraints,
                      method='SLSQP')
    return result.x

# 使用示例
X_train = np.array([[1, 2], [2, 3], [3, 3], [2, 1], [3, 2], [4, 1]])
y_train = np.array([1, 1, 1, -1, -1, -1])

alpha_opt = train_svm_dual(X_train, y_train)
print(f"优化后的拉格朗日乘子: {alpha_opt}")
print(f"支持向量索引: {np.where(alpha_opt > 1e-5)[0]}")
```

### 3. 核函数：低维到高维的魔术

#### 概念说明

**核函数**是SVM最精妙的设计。它允许我们在**不显式计算高维特征**的情况下，实现数据在高维空间中的内积运算。常见的核函数包括：

- **线性核**：K(x, z) = x·z（等价于原始空间）
- **多项式核**：K(x, z) = (γx·z + r)^d
- **RBF（高斯）核**：K(x, z) = exp(-γ‖x - z‖²)
- **Sigmoid核**：K(x, z) = tanh(γx·z + r)

**核函数的本质**：它定义了一个隐式的特征映射 φ(x)，使得 K(x, z) = φ(x)·φ(z)。这个映射可能将数据映射到无穷维空间（如RBF核）。

#### 原理解析

考虑一维数据：x = [-3, -2, -1, 0, 1, 2, 3]，标签为：y = [-1, -1, 1, -1, 1, 1, -1]。数据在一维空间线性不可分。

如果我们定义映射 φ(x) = (x, x²)，数据变为二维：(-3, 9), (-2, 4), (-1, 1), (0, 0), (1, 1), (2, 4), (3, 9)。此时存在一个椭圆超平面将两类分开。

**RBF核的几何意义**：它衡量两个样本的“相似度”，相似度随距离增大指数衰减。在无限维空间中，RBF核可以拟合任意复杂的决策边界。

#### 代码示例：不同核函数的对比

```python
import numpy as np
from sklearn.svm import SVC
from sklearn.datasets import make_circles
import matplotlib.pyplot as plt

# 生成非线性数据（同心圆）
X, y = make_circles(n_samples=200, noise=0.1, factor=0.5, random_state=42)
y = np.where(y == 0, -1, 1)  # 转换为-1/1标签

# 不同核函数的SVM
kernels = ['linear', 'poly', 'rbf', 'sigmoid']
fig, axes = plt.subplots(2, 2, figsize=(10, 10))

for ax, kernel in zip(axes.ravel(), kernels):
    model = SVC(kernel=kernel, gamma='scale', C=1.0)
    model.fit(X, y)
    
    # 绘制决策边界
    xx, yy = np.meshgrid(np.linspace(-1.5, 1.5, 100), np.linspace(-1.5, 1.5, 100))
    Z = model.decision_function(np.c_[xx.ravel(), yy.ravel()])
    Z = Z.reshape(xx.shape)
    
    ax.contourf(xx, yy, Z, levels=[-1, 0, 1], alpha=0.3, colors=['blue', 'gray', 'red'])
    ax.scatter(X[:, 0], X[:, 1], c=y, cmap='bwr', edgecolors='k')
    ax.set_title(f'Kernel: {kernel}')
    ax.set_xlim(-1.5, 1.5)
    ax.set_ylim(-1.5, 1.5)

plt.tight_layout()
plt.show()

# 输出各模型准确率
for kernel in kernels:
    model = SVC(kernel=kernel, gamma='scale', C=1.0)
    model.fit(X, y)
    acc = model.score(X, y)
    print(f"{kernel} kernel accuracy: {acc:.3f}")
```

## 知识依赖说明

理解本文内容需要以下前置知识：

1. **线性代数基础**：向量内积、范数、线性变换。这些是理解超平面表示和核函数映射的基础。
2. **凸优化基础**：拉格朗日乘子法、对偶问题。需要理解如何将约束优化转化为无约束优化。
3. **Python编程**：NumPy基础和Scikit-learn的基本使用。

**推荐学习路径**：
- 如果对线性代数不熟悉，建议先复习“向量空间”和“内积空间”概念
- 如果对凸优化感到陌生，可以从“拉格朗日乘子法”开始，理解如何求解约束优化问题

## 实践建议

1. **从线性SVM开始调试**：在真实项目中，先用线性SVM建立基线。如果线性SVM表现不佳，再尝试RBF核。这有助于理解数据是否线性可分。

2. **重视特征缩放**：SVM对特征尺度敏感。**务必对数据进行标准化**（如使用 `StandardScaler`），否则RBF核的γ参数会失效。

3. **网格搜索调参**：对于RBF核SVM，关键参数是 **C**（正则化强度）和 **γ**（核宽度）。使用 `GridSearchCV` 进行系统调参，通常C的范围为[0.1, 1000]，γ的范围为[0.0001, 10]。

4. **小样本优先使用SVM**：当样本数小于10000且特征维度较高时，SVM往往优于神经网络。这是SVM的“甜蜜区”。

5. **利用支持向量分析数据**：检查训练好的SVM的支持向量——如果支持向量占比过高（>50%），说明数据噪声大或C参数不合适，需要调整。
